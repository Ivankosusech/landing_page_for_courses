Программа курса

Чему вы научитесь

Пошаговая программа от основ до продвинутых технологий. 6 месяцев интенсивного обучения с практикой

1
Недели 1-4

Основы HTML & CSS 4 недели

Изучите фундамент веб-разработки. Научитесь создавать структуру страниц и стилизовать их по современным стандартам.

  • Семантическая вёрстка HTML5
  • Flexbox и Grid Layout
  • Адаптивная вёрстка (Responsive)
  • CSS-анимации и переходы
  • Методология БЭМ
  • Проект: Лендинг компании
2
Недели 5-10

JavaScript для начинающих 6 недель

Освойте главный язык веб-разработки. От переменных до асинхронных запросов.

  • Основы синтаксиса JS
  • Работа с DOM
  • События и обработчики
  • ES6+ (стрелки, деструктуризация)
  • Fetch API и работа с сервером
  • Проект: Интерактивный калькулятор
3
Недели 11-16

React и современные фреймворки 6 недель

Изучите самую востребованную библиотеку для создания интерфейсов.

  • Компоненты и JSX
  • Хуки (useState, useEffect)
  • React Router
  • State Management (Redux)
  • Работа с API
  • Проект: Интернет-магазин
4
Недели 17-22

Backend на Node.js 6 недель

Научитесь создавать серверную часть приложений и работать с базами данных.

  • Node.js и Express
  • REST API
  • Базы данных (MongoDB, PostgreSQL)
  • Аутентификация и авторизация
  • Деплой на сервер
  • Проект: Социальная сеть
5
Недели 23-26

Дипломный проект 4 недели

Создайте полноценное приложение под руководством ментора для портфолио.

  • Выбор темы проекта
  • Проектирование архитектуры
  • Разработка в команде
  • Code Review
  • Презентация проекта
  • Защита диплома

FAQ

Частые вопросы

Ответы на самые популярные вопросы о курсе

Нужен ли опыт программирования для начала?

+

Нет, курс рассчитан на полное отсутствие опыта. Мы начинаем с самых основ и постепенно переходим к сложным темам. Главное — ваше желание учиться и уделять время практике (минимум 10-15 часов в неделю).

Сколько времени нужно уделять обучению?

+

Рекомендуем уделять обучению 15-20 часов в неделю. Лекции можно смотреть в удобное время, они остаются в записи. Домашние задания нужно сдать до следующего модуля.

Что если не успеваю пройти курс?

+

У вас есть доступ к материалам на 12-18 месяцев (в зависимости от тарифа). Можно учиться в своём темпе. Если нужны дополнительные сроки — обсудим индивидуально с куратором.

Помогаете ли с трудоустройством?

+

Да! На тарифах «С ментором» и «VIP» мы помогаем составить резюме, подготовить портфолио и проводим mock-собеседования. У нас есть партнёры-работодатели, которым рекомендуем лучших выпускников. 92% наших студентов находят работу в течение 3 месяцев после выпуска.

Можно ли оплатить в рассрочку?

+

Да, мы предлагаем беспроцентную рассрочку на 12 месяцев от наших партнёров. Первый платеж через месяц после начала обучения. Также принимаем оплату картой и криптовалютой.
