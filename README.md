## 📋 О проекте

**WebPro** — это одностраничный лендинг для образовательного курса по веб-разработке. Сайт разработан для привлечения студентов и содержит все необходимые элементы для конверсии:

- 🎯 Герой-секция с УТП и призывом к действию
- 📚 Детальная программа курса по модулям (5 этапов, 6 месяцев)
- 👨‍🏫 Блок о преподавателе с опытом и достижениями
- 💬 Отзывы реальных выпускников с рейтингом
- 💰 Гибкие тарифные планы с акционными ценами
- ❓ Раздел FAQ с интерактивным аккордеоном
- 📩 Футер с навигацией и контактами

🔗 **Посмотреть демо**: [https://ivankosusech.github.io/landing_page_for_courses/](https://ivankosusech.github.io/landing_page_for_courses/)

## ✨ Особенности

- ✅ **Полностью адаптивный дизайн** — корректное отображение на мобильных, планшетах и десктопах
- ✅ **Семантическая вёрстка** по стандартам HTML5
- ✅ **Оптимизированный код** — чистая структура, комментарии, легко поддерживать
- ✅ **Плавная навигация** по якорным ссылкам с учётом фиксированного хедера
- ✅ **Интерактивные элементы**:
  - Мобильное бургер-меню
  - Раскрывающиеся вопросы в FAQ
  - Плавный скролл к секциям
- ✅ **Готов к деплою** — работает на GitHub Pages без дополнительной настройки
- ✅ **Без внешних зависимостей** — только чистый код, быстрая загрузка
